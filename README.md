# Vallex4UMR

This repository contains all the materials of the **Vallex4UMR** project.

The Vallex4UMR file is generated by running the `converter.py` script as follows:

- Step 1: generate the base file

```
python3 converter.py --wordnet path/to/latin/wordnet --mapping latvallex2_umr_mapping.tsv --vallex path/to/latin/vallex2
```

- Step 2: add frames for _sum_ and _habeo_, as described in `sum_habeo_frames.csv`
```
python3 append_sum_frames.py
```

### Resources

The required files are available on the GitHub of the CIRCSE Research Center:

1. Latin WordNet: [LiLa_LatinWordnet.csv](https://github.com/CIRCSE/latinWordnet-revision/blob/master/LiLa_LatinWordnet.csv)

2. Latin Vallex 2.0: [export_LV.tsv](https://github.com/CIRCSE/Latin_Vallex2.0/blob/master/export_LV.tsv)
